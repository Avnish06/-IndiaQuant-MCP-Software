# IndiaQuant MCP Server

A real-time Indian stock market AI assistant using Model Context Protocol (MCP) and 100% free APIs. Built for the IndiaQuant Take-Home Assignment.

## Features

- **Live Market Data**: Real-time NSE/BSE prices via `yahoo-finance2`.
- **Options Intelligence**: Built-in Black-Scholes Greeks (Delta, Gamma, Theta, Vega) implemented from scratch.
- **AI Signals**: Trading signals generated by combining technical indicators (RSI, MACD, BB) with news sentiment.
- **Virtual Trading**: Persistent SQLite-based portfolio to track virtual trades and P&L.
- **Unusual Activity**: Detection of OI spikes and high volume-to-OI ratios in the options chain.

## 🛠️ Architecture

### 1. Market Data Engine (`marketData.ts`)
Handles all external data fetching from Yahoo Finance. Implements symbol formatting for NSE indices and stocks.

### 2. Options Analyzer (`optionsAnalyzer.ts`)
Manages derivatives data. Includes a custom **Max Pain** calculator and integrates the `blackScholes.ts` utility for real-time Greeks calculation.

### 3. AI Signal Generator (`signalGenerator.ts`)
Uses the `technicalindicators` library and NewsAPI to derive a confidence score.
- **Technicals (50%)**: RSI (oversold/overbought) and MACD histogram trends.
- **Sentiment (50%)**: Keyword-based analysis of recent headlines.

### 4. Portfolio Manager (`portfolioManager.ts`)
Maintains a local SQLite database (`data/portfolio.db`) for tracking virtual cash (initial ₹10 Lakhs) and positions.

### 5. MCP Layer (`index.ts`)
The bridge between our quant logic and Claude. Implements the MCP protocol for tool registration and execution.

## 🚀 Setup & Installation

### Prerequisites
- Node.js (v18+)
- NewsAPI Key (Get free at [newsapi.org](https://newsapi.org/))

### Configuration
Create a `.env` file in the root directory:
```env
NEWS_API_KEY=your_news_api_key_here
```

### Installation
```bash
npm install
npm run build
```

### Usage with Claude Desktop
Add this to your Claude Desktop configuration file (`%APPDATA%\Claude\claude_desktop_config.json`):
```json
{
  "mcpServers": {
    "indiaquant": {
      "command": "node",
      "args": ["C:/Users/Lenovo/MCP-VectorAssignment/dist/index.js"],
      "env": {
        "NEWS_API_KEY": "your_news_api_key"
      }
    }
  }
}
```

## 📊 10 Required Tools

1. `get_live_price`: Real-time quote for any symbol.
2. `get_options_chain`: Full chain with Max Pain calculation.
3. `analyze_sentiment`: Headlines and score.
4. `generate_signal`: BUY/SELL/HOLD with confidence.
5. `calculate_greeks`: Pure math Black-Scholes Greeks.
6. `detect_unusual_activity`: Alerts for OI volatility.
7. `place_virtual_trade`: Buy/Sell with virtual cash.
8. `get_portfolio_pnl`: Real-time P&L and risk score.
9. `scan_market`: Find stocks matching RSI criteria.
10. `get_sector_heatmap`: Sector-wise performance.
